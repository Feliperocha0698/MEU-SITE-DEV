# 🌐 Meu Site Pessoal

Este é o repositório do meu site pessoal, desenvolvido em **HTML5, CSS3 e JavaScript**.  
Aqui você encontra páginas institucionais, portfólio de projetos, serviços oferecidos e formas de contato.

---

## 📂 Estrutura do projeto
- `index.html` → Página principal  
- `sobre-mim.html` → Informações sobre mim  
- `portfolio.html` → Meus projetos  
- `servicos.html` → Serviços oferecidos  
- `contato.html` → Página de contato  
- `assets/css` → Arquivos de estilo (reset.css, style.css, media_queries.css)  
- `assets/img` → Imagens utilizadas  
- `js` → Scripts JavaScript (script.js, pesquisa.js)
